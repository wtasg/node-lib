Defined in: [src/strings.ts:599](https://github.com/wtasg/node-lib/blob/1ee11d15e0962dff67357405c5de56cb28136fb5/src/strings.ts#L599)

LastIndexByte returns the index of the last instance of the byte c in s,
or -1 if c is not present in s.

## Parameters

### s

`string`

Input string.

### c

`number`

Byte value (0–255) to search for.

## Returns

`number`

Index of last occurrence, or -1.
