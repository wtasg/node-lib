ToUpperSpecial returns a copy of the string s with all Unicode letters mapped
to their upper case using the rules of the special case c (locale).

## Parameters

### c

`string`

Locale identifier (e.g. "tr", "az").

### s

`string`

Input string.

## Returns

`string`

Locale-upper-cased string.
